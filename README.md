# v-button

v-button のソフトウェア


# インストール

```sh
sudo apt update
sudo apt install ruby
curl -sL https://deb.nodesource.com/setup_8.x | sudo -E bash -
sudo apt-get install -y nodejs
curl -L https://github.com/realglobe-Inc/v-button/archive/master.zip > v-button-master.zip
unzip v-button-master.zip
cd v-button-master
sudo make install
sudo reboot
```
